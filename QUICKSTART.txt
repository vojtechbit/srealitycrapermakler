📋 CO POTŘEBUJEŠ:
  ✓ Python 3.8+ (stáhni z python.org)
  ✓ Terminal/Příkazový řádek

🚀 RYCHLÉ SPUŠTĚNÍ (3 KROKY):

1️⃣  Otevři terminál ve složce sreality_scraper
   
   macOS/Linux:
   - Pravé tlačítko na složku → "Services" → "New Terminal at Folder"
   - nebo: cd /Users/vojtechbroucek/Desktop/sreality_scraper
   
   Windows:
   - Shift + pravé tlačítko na složku → "Otevřít PowerShell zde"

2️⃣  Nainstaluj závislosti (jen poprvé):
   
   pip3 install -r requirements.txt
   
   (nebo: pip install -r requirements.txt)

3️⃣  Spusť scraper:
   
   python3 sreality_scraper.py
   
   (nebo: python sreality_scraper.py)
