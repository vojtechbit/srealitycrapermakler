   2. Klikni "Generate new token" → "Classic"
   3. Dej mu jméno: "sreality-scraper"
   4. Vyber "repo" oprávnění
   5. Zkopíruj token (UKAŽ JEN 1x!)

nebo

   2. Přidej na GitHub: https://github.com/settings/keys
   3. Změň remote URL:

🎯 PRÁCE S CLAUDE CODE:

Po nahrání na GitHub:

   git clone https://github.com/vojtechbit/srealitymaklerscraper.git

2. Otevři v Claude Code:
   - Spusť Claude Code
   - File → Open Folder
   - Vyber složku s projektem

3. Claude Code vidí celý projekt a můžeš s ním:
   ✓ Upravovat soubory
   ✓ Spouštět scraper
   ✓ Vyvíjet nové funkce
   ✓ Testovat změny

4. Synchronizace změn:
   git add .
   git commit -m "Změny z Claude Code"
   git push

   git push
